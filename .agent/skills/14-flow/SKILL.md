# Skill 14: Delivery Flow

## Standard Flow

1. Define boundary and acceptance behavior.
2. Identify variation/risk points.
3. Implement smallest useful slice.
4. Verify with targeted tests.
5. Record follow-up debt explicitly.

## Mandatory Rules

- Split multi-feature work into independent increments.
- Avoid heavyweight planning for trivial tasks.
- Keep implementation traceable to acceptance criteria.
- Summarize done scope and residual risk.
