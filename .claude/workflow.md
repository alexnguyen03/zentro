# Development Workflow

## When fixing bugs:

1. Identify root cause
2. Do NOT fix immediately
3. Confirm hypothesis
4. Apply minimal fix
5. Suggest test case

## When refactoring:

1. Preserve behavior
2. Improve readability
3. Reduce complexity
4. Do not change API unless asked

## When adding feature:

1. Identify layer to modify
2. Avoid cross-layer violation
3. Keep change isolated
